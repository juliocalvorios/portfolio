# Julio Calvo - Portfolio

Personal portfolio website designed as a vintage newspaper. Built with React and Tailwind CSS.

## Tech Stack

- React 18
- Tailwind CSS
- Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design Concept

The portfolio is designed to look like a vintage newspaper (The Calvo Chronicle), with:

- Traditional newspaper typography and layout
- Multiple sections: Front Page, Selected Works, About, Classifieds
- Editorial-style project presentations
- Ornamental dividers and classic styling
